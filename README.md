- Grouped together the most popular 1000 names each year and produced a graph showing the percentage of given names that fall in the top 1000 over time
- Shown how the popularity of specific names vary over time
- Produced 2 bar graphs showing the most popular first letter in names over time, for male and female

```python
years = range(1880, 2019)
pieces = []

for year in years:
    path = '/home/jake_ockerby/Python/names/yob%d.txt' % year
    frame = pd.read_csv(path, names=['name','sex','births'])
    frame['year'] = year
    pieces.append(frame) # reading all files from 1880-2018 into dataframes, then storing
                         # the dataframes in a list
        
names = pd.concat(pieces, ignore_index=True)# concatenates all dataframes in the list into one dataframe
```

```python
total_births = names.pivot_table('births', index='year', columns='sex', aggfunc=sum)
total_births.tail()
```

```python
def add_perc(group):
    births = group.births.astype(float)
    group['percentage'] = (births/births.sum())*100
    return group

names = names.groupby(['year','sex']).apply(add_perc)
names # creating a column for birth names as a percentage of total births by year and sex
```
